组网


这篇指南旨在帮助第一次使用 CloudIn 平台进行组网且有一定的平台使用经验的用户，帮助用户利用已经创建的主机资源进行简单的组网操作，查看网络拓扑图。

适用场景

本文假定用户已经创建了主机，第一次进行组网操作。假设我们的任务是在“北京”节点创建两台主机，将他们分别放在不同的二层网络（公网子网）当中，启动 DHCP 服务，这两个二层网络需要通过一个路由器在三层上进行联通。

操作步骤

第一步：搭建两个二层网络（公网子网）

登录控制台并选择地域“北京”，点击左边的导航栏中的“网络￫子网”，在右边的显示区域中点击“创建子网”按钮;

在弹出的页面当中输入子网的名称，选择子网的类型（这里选择公网子网），输入网络地址，选中启用 DHCP ，点击“确认”。

第二步：将创建的主机分别放置于这两个网络当中

其它部分已经描述了如何为主机修改网络配置，这里将不再进行描述。这里在加入网络时，选择需要加入的网络即可。这里，我们建立了两个二层网络，需要将这两台主机绑定到不同的网络当中去。

第三步：创建一个路由器

点击左边的导航栏中的“网络￫路由器”，在右边的显示区域中点击“创建路由器”;

输入名称，选择需要创建的数量为1，开启公网网关。（如果您需要连接到公网中，选择开启）

第四步：给路由器关联子网

点击左边的导航栏中的“网络￫路由器”，在右边的显示区域中选中需要操作的路由器，点击“关联子网”;

在弹出的页面中选择需要关联的子网，点击“确定”。
