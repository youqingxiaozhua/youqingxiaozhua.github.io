资源创建


本指南旨在帮助初次使用 CloudIn 的用户，帮助用户能够创建主机，完成类似于创建远程服务器的功能。这是一篇指导用户入门的指南，如果用户有更深层次的需求，请参考 CloudIn 云服务器中的对应部分。

适用场景

这篇指南假定用户在“北京”节点创建一个2核4G内存的具有 centos 6.3系统的主机（主机计费方式为实时计费），挂载一个100GB的存储型硬盘，并为主机绑定一个带宽为1Mbps、按带宽计费的公网IP ，使用用户名密码登录主机。

以下为具体操作步骤：

1. 创建主机

创建一个2核4G内存的主机

登录控制台，点击“地域”，下拉选择“北京”，点击左侧导航栏中的“计算￫主机”，点击右侧显示区域的“创建主机”按钮；

在跳转的页面中选择 centos 6.3，点击“下一步”按钮；

在显示的页面中的 CPU 栏选择2核，内存栏选择4G，点击“下一步”按钮；

在显示的页面中点击“下一步”按钮；

在显示的页面中输入主机名称， SSH 登录方式选择“密码”登录，在密码栏输入需要设定的密码，在右下角会显示当前所选资源所需要的费用，点击“创建”。

2.创建并挂载硬盘

登录控制台并选择地域“北京”，点击左侧导航栏中的“计算￫硬盘”，在右侧显示界面点击“创建硬盘”按钮；

在显示的页面中，在名称栏输入需要设定的硬盘的名称，如“创建硬盘”，硬盘数量选定1个，硬盘的容量向右拖到100GB，存储方式选择存储型，在右下脚会显示当前所选硬盘资源所需要的费用，点击“确认”。

3.申请IP并绑定

登录控制台并选择地域“北京”，点击左侧导航栏中的“网络￫公网IP ”，在右侧显示界面点击“申请”按钮；

输入公网IP 名称，选择申请的公网IP 数量为1，选择申请的公网IP 的带宽为1Mbps，以及计费模式选择按带宽计费，在右下角会显示当前所选资源所需要的费用，点击确定；

登录控制台并选择地域“北京”，选择左侧的导航栏中的“网络￫公网IP ”，在右侧的显示界面中选择需要绑定的公网IP ，点击“分配到主机”；

在弹出的页面中的云主机栏，下拉选择被绑定的主机，点击“确认”。

4.远程连接主机

Mac和Linux用户打开终端，输入ssh -l <远程主机用户名> <远程主机ip> ， 然后输入密码即可。

Windows用户可以使用Putty等第三方客户端。
