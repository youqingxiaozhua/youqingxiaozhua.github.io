设置安全组规则


创建安全组规则

所有安全组规则默认都是禁止的，请根据您的需要自行添加允许的规则，目前暂时只支持上行规则。

点击安全组名称进入安全组详情，切换到规则详情标签，点击“添加规则”按钮，填写该条规则的协议、起始端口、结束端口、源IP，支持批量添加 。

 

修改安全组规则

在规则详情标签页下，直接点击需要修改的地方，该安全组即进入编辑状态，点击右侧的保存按钮。（支持批量保存）

 

删除安全组规则

在规则详情标签页下，点击需要删除的安全组规则右侧的关闭按钮，即可删除该条规则。您也可以利用安全组规则前方的复选框同时删除多条安全组规则。

注意：处于编辑状态下的规则不支持批量删除。
