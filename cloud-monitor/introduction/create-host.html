创建主机

登录 CloudIn 控制台，并选择“地域”；选择“计算￫主机”，点击“创建”按钮，这时会弹出创建主机的向导：

1. 选择镜像

2. 选择配置

3. 挂载空闲硬盘（可选）

4. 选择网络

5. 设置安全组

6. 给主机命名

（其中主机名限制在20个字符以内，密码至少8位，并包括大小写字母及数字）

7. 设置登陆方式

8. 完成创建

通常创建过程在10秒钟左右
