监控报警


这篇指南旨在帮助第一次使用 CloudIn 平台云安全服务的用户，帮助用户使用云监控产品。

适用场景

本文假定用户已经创建了主机，希望了解资源的使用情况。假设我们的任务是创建一个对主机 CPU 利用率进行监控的告警规则，绑定到已经创建的主机上，并将告警信息发送给预先设置好的用户。

其中，告警规则设置为在监控周期内 CPU 利用率三次达到90%及以上，内存利用率2次达到90%及以上；将“通知条件”设置为“告警时”，“通知方式”设置为“邮件”以及短信。

1.创建通知组，添加成员；

登录 CloudIn 控制台并选择“地域”；选择左侧导航栏的“监控告警￫通知组”，点击右侧"通知组列表"中的“创建通知组”；

在“创建通知组”页面中填写名称，点击“确认”。

在"通知组列表"中点击需要添加成员的组 ID ，进入成员列表；

在成员列表中点击“添加成员”；

在“创建成员”页面中填写“姓名”、“手机号”以及“电子邮箱”，点击“确认”。

2.创建告警

在左侧导航栏选择“监控告警￫告警策略”，在右侧主页面中点击“创建告警”；

在“创建告警”页面中输入名称，如“ CloudIn 告警”；选择“资源类型”：主机；选择“监控周期”：5分钟；点击“下一步”；

设置“触发条件”，点击“下一步”；

选择需要应用该告警规则的资源，点击“下一步”；

“通知条件”选择“告警时”，“通知列表”中下拉选择通知组，“通知方式”选择“邮件”以及“短信”，点击创建；

如果您有更多问题, 发送邮件 或者 提工单 反馈给我们。
